# Animevibez v4

Animevibez is a clean and modern anime-themed web project focused on simplicity, design, and user experience.  
It includes pages like Home, FAQ, Contact, Help, and Watchlist — all connected and styled using Tailwind CSS.

---

## Version
**Current version:** v4  
This version includes several updates, UI fixes, and code cleanups done by Hiro Hudson.

---

## Project Overview
Animevibez lets users browse and save anime to a local watchlist, toggle between dark and light themes,  
and search anime titles easily. It’s built for smooth performance and responsive viewing on any device.

**Main features:**
- Dark/Light theme toggle with saved preference  
- Real-time anime search  
- LocalStorage-powered watchlist (add/remove anime)  
- Responsive design for all devices  
- Organized pages for FAQ, Help, and Contact  
- Genre navigation system  
- Animation and transition effects for better UX
